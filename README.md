# Quiz Application

## Overview
The Quiz Application is a console-based Java application that allows users to answer multiple-choice questions and receive a score based on their performance. The project is designed to demonstrate Core Java concepts such as Object-Oriented Programming (OOP), arrays, collections, loops, methods, and exception handling.

## Features
- Multiple-choice quiz questions
- User-friendly console interface
- Automatic score calculation
- Input validation
- Displays final results after quiz completion

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Collections Framework
- Exception Handling
- Scanner Class

## How to Run

1. Clone the repository:
   git clone <repository-url>

2. Navigate to the project directory:
   cd QuizApplication

3. Compile the Java files:
   javac *.java

4. Run the application:
   java Main
